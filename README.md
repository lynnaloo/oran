### Running Oranjs
To run Oranjs on your machine, open up your terminal, go to the directory you'd like to install Oranjs and cover the following steps: 
`$ git clone https://github.com/lynnaloo/oranjs.git && cd oranjs`
`$ npm i`
`$ node index.js`

Voila!
